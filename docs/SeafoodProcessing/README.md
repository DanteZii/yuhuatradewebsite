## BreadedOyseters
**形态**：    冷冻品 <br>
**保存条件**： 18℃ 以下<br>
**规格**：    500g/盘/袋<br>
**包装形式**： 10袋/箱<br>
**加工流程**： 牡蛎原料解冻-沾干面粉-面糊-面包渣-速冻-包装<br>
**调理**：    无需解冻，冷冻状态下，直接放入175～180℃的油温中，油炸4～4分30秒即可
