## 我们是谁?
烟台裕华工贸有限公司成立于2005年，是一家集全球一手货源采购、加工及内外销业务多方位发展的工贸一体公司。历经10余年的发展公司在各项业务中积累了丰富的经验，产品与各项服务也得到了业界的一致好评。“百尺竿头更进一步”，裕华愿与全球同仁砥砺前行、不断开拓美好的未来！
